# Database Setup

This project uses a **PostgreSQL** database and handles connections through environment configs and Docker Compose.

---

## Configuration

Database connection is managed via environment variables:

| Variable           | Description               |
|--------------------|---------------------------|
| `DB_HOST`          | Hostname (default: `db`)  |
| `DB_PORT`          | Port (default: `5432`)    |
| `DB_USER`          | Username                  |
| `DB_PASSWORD`      | Password                  |
| `DB_NAME`          | Database name             |
