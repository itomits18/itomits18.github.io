# Prerequisites
We recommend you to uses the **prerequisites** that we provided
<Steps>

1. Golang
   This project use **Golang** Programming language, you can install from link below
   ```js
   install golang from 'https://go.dev/doc/install';
   ```
2. IDE 
   This project use **Goland** from Jetbrains, you can install from link provided
   ```js
   install Goland IDE from 'https://www.jetbrains.com/go/download';
   ```
3. Docker
   This project use **Docker Dekstop** as it's launch container, you can follow the installation instruction from the link provided
   ```js
   install Docker Dekstop from 'https://docs.docker.com/desktop/setup/install';
   ```
4. Postman
   This project use **Postman** to organize API's project, you can see the instruction from the link provided
   ```js
   install Postman from 'https://www.postman.com/downloads';
   ```
</Steps>
