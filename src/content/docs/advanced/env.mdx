## Environment Variables

We use a `.env` file to store sensitive and environment-specific values. These values are loaded into the application using a Go library such as [`github.com/joho/godotenv`](https://github.com/joho/godotenv).

## Application Settings

| Key           | Description                                | Example          |
|---------------|--------------------------------------------|------------------|
| `APP_PORT`    | Port the application will run on           | `8080`           |
| `TIMEZONE`    | App timezone                               | `Asia/Jakarta`   |
| `APP_STATUS`  | Current environment (`local`, `prod`, etc) | `local`          |

## Database Configuration

> The app supports multiple DB engines: SQLite, PostgreSQL, MySQL

### SQLite
| Key            | Description                    | Example                  |
|----------------|--------------------------------|--------------------------|
| `SQLITE_PATH`  | Path to SQLite file            | `databases/database`     |

### PostgreSQL
| Key               | Description                     | Example      |
|-------------------|---------------------------------|--------------|
| `POSTGRES_USER`   | Username for PostgreSQL         | `postgres`   |
| `POSTGRES_PASSWORD` | Password for PostgreSQL      | `pass`       |
| `POSTGRES_DB`     | Database name                   | `db`         |
| `POSTGRES_PORT`   | Port used                       | `9920`       |
| `POSTGRES_HOST`   | Host (usually `localhost` or `db`) | `localhost`  |

### MySQL
| Key            | Description                     | Example      |
|----------------|---------------------------------|--------------|
| `MYSQL_USER`   | Username for MySQL              | `mysql`      |
| `MYSQL_PASSWORD` | Password for MySQL            | `pass`       |
| `MYSQL_DB`     | Database name                   | `db`         |
| `MYSQL_PORT`   | Port used                       | `3307`       |
| `MYSQL_HOST`   | Host                            | `localhost`  |

## Example `.env`

```env
APP_PORT=8080
TIMEZONE=Asia/Jakarta
APP_STATUS=local
DB_DRIVER=postgres

SQLITE_PATH=databases/database

POSTGRES_USER=postgres
POSTGRES_PASSWORD=pass
POSTGRES_DB=db
POSTGRES_PORT=9920
POSTGRES_HOST=localhost

MYSQL_USER=mysql
MYSQL_PASSWORD=pass
MYSQL_DB=db
MYSQL_PORT=3307
MYSQL_HOST=localhost
